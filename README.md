# Network Technologies - Interactive Activities

Fun, engaging activities for CCEA GCSE Digital Technology - Network Technologies topic.

## Activities Included

1. **LAN vs WAN Sorting Game** - Drag and drop timed game 
2. **Internet vs WWW vs IoT Challenge** - Quick-fire category matching
3. **Communication Tech Rocket Race** - Answer questions to launch your rocket
4. **Network Hardware Builder** - Build networks and answer questions
5. **Topology Maze Runner** - Navigate through network topologies 

## How to Use

### Option 1: Open Locally
Simply open `index.html` in any web browser - all activities work offline! 

### Option 2: Host on GitHub Pages (Recommended for sharing with students)

1. Create a GitHub account at github.com
2. Click "New repository"
3. Name it `network-activities`
4. Upload all files from this folder
5. Go to Settings → Pages
6. Select "main" branch and click Save
7. Your site will be live at: `https://yourusername.github.io/network-activities/`
